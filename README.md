# Score Engine (Internal Calculator) 📊

Score Engine is a modern Android application designed for Vignan's university students to manage their academic performance. It helps you track your internal assessment marks and calculate your Semester Grade Point Average (SGPA) with ease. (R22C24 Regulation)

## 🚀 Features

- **Internal Marks Tracker**: 
  - Add, edit, and delete subjects.
  - Track marks for different internal assessments.
  - Persistent storage using **Room Database**.
  - Modern UI with `RecyclerView` and `FloatingActionButton`.
- **SGPA Calculator**:
  - Dynamically generate input fields based on the number of subjects.
  - Calculate SGPA based on Grade Points and Credits.
  - Real-time result display with a clean card interface.
- **Modern UI/UX**:
  - Smooth transitions and animations.
  - Navigation Drawer for easy access to features.
  - Material Design 3 components.
  - Back-press handling to prevent accidental exits.

## 🛠️ Built With

- **Language**: Java
- **Database**: Room (SQLite)
- **UI Components**: 
  - Material Design Components (Google)
  - DrawerLayout & NavigationView
  - RecyclerView
  - ValueAnimator for smooth UI transitions
- **Architecture**: Android Jetpack (AppCompat, Activity, Transition)
